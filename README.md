# 🧠 Animal Image Classification using CNN

## 📌 Overview

This project is an animal image classification system built using a **Convolutional Neural Network (CNN)**. The application allows users to upload an animal image and predicts its class through a Streamlit web interface.

---

## ✨ Features

- Upload animal images
- Predict animal classes using CNN
- User-friendly Streamlit interface
- Real-time image prediction

---

## 🛠️ Technologies Used

- Python
- TensorFlow / Keras
- Streamlit
- OpenCV
- NumPy
- Matplotlib

## 🚀 Installation

Clone this repository:

```bash
git clone https://github.com/deviagung01/animal-image-classification-cnn.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
